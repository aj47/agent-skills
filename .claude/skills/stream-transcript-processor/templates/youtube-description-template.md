## Section Guidelines

### Hook (First 2 Lines)
- Most important - shows in search results
- Start with emoji for visual appeal
- Include main keyword/tool name
- Create curiosity or promise value

**Examples:**
```
üöÄ Claude Haiku just solved Wordle in 43 seconds - a new benchmark record!

Testing different browser automation MCPs to find the fastest AI agent setup.
```

```
üî• I found an AI coding tool that's 10x faster than Cursor

Live demo and benchmark comparison of the latest AI coding assistants.
```

### Timestamps Section
- Start with `0:00 - Intro`
- Key moments every 2-5 minutes
- Use action words: "Testing", "Setting up", "Results", "Comparing"
- Include the payoff/result timestamp
- Max 10-12 timestamps for readability

**Format:**
```
‚è±Ô∏è TIMESTAMPS
0:00 - Intro
2:15 - Setting up Playwright MCP
5:30 - First benchmark attempt
8:45 - Testing Playrider MCP
12:00 - Haiku vs Sonnet comparison
15:30 - NEW RECORD: 43 seconds!
18:00 - Final thoughts
```

### Tools & Links Section
- Only include tools actually discussed
- Use official URLs
- Order by relevance/mention frequency
- Include affiliate links where applicable

**Format:**
```
üîó TOOLS MENTIONED
‚Ä¢ Cursor: https://cursor.com
```

### Social Links Section
- Keep consistent across all videos
- Update Discord/community links as needed

### Hashtags
- Place at end of description
- 5-8 relevant hashtags
- Include tool names as hashtags
- Mix broad (#AI #Coding) and specific (#ClaudeCode #PlaywrightMCP)

## SEO Keywords to Include

### Primary (use in title + first 2 lines)
- AI coding
- AI tools
- [Specific tool name]
- Live coding
- Tutorial/Demo/Benchmark

### Secondary (use in description body)
- Programming
- Developer tools
- Automation
- Workflow
- Productivity

### Long-tail (for timestamps/context)
- "[Tool] tutorial 2024"
- "best AI coding assistant"
- "[Tool] vs [Tool]"
- "how to use [Tool]"
